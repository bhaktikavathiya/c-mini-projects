# Calculator

## About

Calculator is a command-line program written in C that performs basic arithmetic operations. The user enters two numbers and an operator, and the program calculates and displays the result. It also handles invalid operators and prevents division by zero.

## Features

* Performs addition, subtraction, multiplication, and division
* Accepts both `*` and `x` for multiplication
* Displays the calculation and result
* Handles invalid operator input
* Prevents division by zero

## Concepts Used

* Variables
* `printf()`
* `scanf()`
* `if`, `else if`, `else`
* Nested `if` statements
* Arithmetic operators (`+`, `-`, `*`, `/`)
* Comparison operators
* Logical operators (`||`)
* Character input (`char`)
* Floating-point numbers (`float`)
* Input validation
