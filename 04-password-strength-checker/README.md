# Password Strength Checker

## About

Password Strength Checker is a command-line program written in C that analyzes a user's password and determines whether it is strong or weak based on basic security rules. The program checks the password for minimum length, uppercase letters, lowercase letters, and numbers.

## Features

* Checks if the password is at least 8 characters long
* Detects uppercase letters
* Detects lowercase letters
* Detects numbers
* Displays whether the password is strong or weak

## Concepts Used

* Variables
* Character arrays (strings)
* `printf()`
* `scanf()`
* `for` loop
* `if`, `else`
* Logical operators (`&&`)
* Comparison operators
* `strlen()`
* Character comparisons using ASCII values
* `<string.h>`
