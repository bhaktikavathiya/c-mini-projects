# Rock Paper Scissors

## About

Rock Paper Scissors is a command-line game written in C where the player competes against the computer. The computer randomly selects Rock, Paper, or Scissors, and the program determines the winner based on the standard game rules.

## Features

* Play against the computer
* Random computer choice
* Input validation
* Displays both the player's and computer's choices
* Detects win, loss, and draw

## Concepts Used

* Variables
* `printf()`
* `scanf()`
* `if`, `else if`, `else`
* Comparison operators
* Logical operators (`&&`, `||`)
* Random number generation (`rand()`, `srand()`)
* `time()`
* Input validation
