# Guess the Number 🎯

## About

This is my first mini project in C.

The computer generates a random number between 1 and 100. The player gets one chance to guess the number. If the guess is incorrect, the game tells the player whether the correct number is higher or lower before revealing the answer.

## Concepts Used

- Variables
- User Input (`scanf`)
- Output (`printf`)
- `if-else` statements
- Random number generation (`rand()`)
- `srand()`
- `time()`

## Current Features

- Generates a random number between 1 and 100
- Takes one guess from the player
- Tells the player whether the guess is correct
- Gives a hint if the guess is too high or too low
- Reveals the correct number
