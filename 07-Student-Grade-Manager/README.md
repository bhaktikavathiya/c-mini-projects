# Student Grade Manager

## About

Student Grade Manager is a command-line program written in C that allows the user to enter a student's name and marks for five subjects. The program calculates the total marks, percentage, assigns a grade based on the percentage, and displays whether the student has passed or failed.

## Features

* Enter student name
* Input marks for five subjects
* Calculate total marks
* Calculate percentage
* Assign grades (A to F)
* Display Pass/Fail result
* Uses arrays and loops for efficient data handling

## Concepts Used

* Variables
* Arrays
* `printf()`
* `scanf()`
* `for` loops
* `if`, `else if`, `else`
* Arithmetic operators
* Comparison operators
* Input validation using field width (`%19s`)
