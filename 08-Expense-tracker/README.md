# Expense Tracker v1

A simple console-based Expense Tracker built in C. This program allows users to record multiple expenses, categorize them, calculate the total expense, and display a neatly formatted summary.

## Features

- Enter up to 20 expenses
- Record expense category and amount
- Validate the number of expenses (1–20)
- Reject zero or negative expense amounts
- Calculate the total expense automatically
- Display a formatted expense summary

## Concepts Used

- Variables
- Arrays
- 2D Character Arrays (Strings)
- Loops (`for`)
- Conditional Statements (`if`, `else`)
- User Input (`scanf`)
- Formatted Output (`printf`)
- Input Validation
