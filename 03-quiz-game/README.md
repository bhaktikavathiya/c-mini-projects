# Quiz Game

## About

Quiz Game is a command-line quiz application written in C where the player answers 10 multiple-choice cricket questions. The program checks each answer, keeps track of the score, and displays the final result with a performance message based on the player's score.

## Features

* 10 multiple-choice cricket questions
* Instant feedback after each answer
* Displays the current score after every question
* Input validation
* Final score and performance message

## Concepts Used

* Variables
* `printf()`
* `scanf()`
* `if`, `else if`, `else`
* Comparison operators
* Logical operators (`&&`)
* Input validation
