# ATM Simulator

## About

ATM Simulator is a command-line program written in C that simulates the basic functions of an ATM. The user can check their account balance, deposit money, withdraw money, or exit the program. The program also validates user input to prevent invalid transactions such as negative deposits, negative withdrawals, and withdrawals that exceed the available balance.

## Features

* Check account balance
* Deposit money
* Withdraw money
* Prevent withdrawal when funds are insufficient
* Validate deposit and withdrawal amounts
* Handle invalid menu choices
* Simple menu-driven interface

## Concepts Used

* Variables
* `printf()`
* `scanf()`
* `if`, `else if`, `else`
* Nested `if` statements
* Comparison operators
* Arithmetic operators
* Input validation
